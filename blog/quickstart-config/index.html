設定ファイルとタイトル

  • tags:
  • lektor
  • quickstart
on 2023-01-17 23:00:00

プロジェクトのルートにある設定ファイルlektor-demo.lektorconfigは、プロジェクト全般に亘る設定値を管理します。

1. タイトルの変更

ヘッダーのタイトルに、設定ファイルへ記述された[project] nameに指定された文字列を表示できるように変更します。

lektor-demo.lektorconfig

[project]
name = lektor-demo
name = 日本語のTitle

templates/layout.html

<body>
  <header>
    <h1>lektor-demo</h1>
    <h1>{{ config.PROJECT.name }}</h1>
