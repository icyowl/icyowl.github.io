レスポンシブ対応

  • tags:

on 23-01-17

タイポグラフィの検討やHtml要素のスタイルを書く前に、モバイルに対応するためのメディアクエリを書きます。

1. メディアクエリ

style.cssでは、 header, footer, pageの幅がwidth:760pxに固定されているので、 max-widht:760pxに変更します。

ブレークポイントでの変更点は次の通りです。

  • root font size 16px -> 14px
  • root line-hight 1.75 -> 1.5
  • body margin: 0 -> 2px
  • header, footer, page { padding: 20px 30px } -> padding: 1rem

assets/static/custom.css (追記)

/* responsive */
header, footer, div.page {
  width: initial;
  max-width: 760px;
}

@media screen and (max-width: 576px) {
  html {
    font-size: 14px;
    line-height: 1.5;
  }
  body {
    margin: 2px;
  }
  header, footer, div.page {
    padding: 1rem;
  }
}

2. Viewport

templates/layout.html (追記)

<meta charset="utf-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<link rel="stylesheet" href="{{ '/static/style.css'|url }}">
<link rel="stylesheet" href="{{ '/static/custom.css'|url }}">
