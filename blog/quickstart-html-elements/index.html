2. レイアウトの調整

前回のサンプル記事を見ると、ヘッダーと記事タイトルの間や、投稿日付の上下のスペースが少し広すぎるようです。heading要素のmarginや、p要素のline-heightがレイアウトに影響しているので修正します。

assets/static/custom.css (追記)

/* layout adjustment */

div.page h2:first-child {
  margin-top: var(--spacing-2);
}

.meta {
  line-height: 1;
}
