ここでは記事ページの表示に係わるHtml要素のスタイルを記述します。

1 Html要素のスタイル

記事内で利用するHtml要素のスタイルをbase.cssへ記述します。

assets/static/base.css (追記)

:root {
  --font-code: Inconsolata, Monaco, Consolas, "Courier New", Courier, system-ui, monospace;
  --gray: #bcc2bf;  /* --gray-3 */
}

/* html elements */
a {
  color: inherit;
}

hr {
  background: var(--gray);
  height: 1px;
  border: 0;
}


ul, ol {
  padding: 0 0 0 2rem;
  margin: 0;
}

li {
  line-height: 1.75;
}

ul {
  list-style-type: circle;
}

ul ul {
  list-style-type: disc;
}

ul ul,
ol ol,
ul ol,
ol ul {
  padding: 0 0 0 1rem;
}

blockquote {
  border-left: 4px solid var(--gray);
  margin-left: 0;
  padding: 0 0 0 .5rem;
}

code {
  font-family: var(--font-code);
  font-size: 0.875rem;
}

p code, li code {
  border: solid 1px var(--gray);
  border-radius: 2px;
  padding: 1px 4px;
}

pre {
  line-height: 1.5;
  border: solid 1px var(--gray);
  border-radius: 4px;
  margin: 1rem 0;
  padding: 1rem;
  overflow-x: scroll;
}

table {
  width: 100%;
  border-collapse: collapse;
}

table, td, th {
  border: 1px solid var(--gray);
}

th {
  text-align: center;
}

td {
  padding: 0 .5rem;
}

img {
  display: block;
  border: none;
  margin: 0 auto;
  width: 100%;
}
